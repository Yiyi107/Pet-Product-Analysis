# Pet-Product-Analysis
This project explores an e-commerce pet supplies dataset from Kaggle. The goal is to analyze product performance, customer preferences, and overall trends in the pet products market.

The dataset contains information on:

title — Product name

quantity — Number of units sold

tradeAmount — Total sales amount

averageStar — Average customer rating

wishedCount — Number of times a product was added to wish lists
Objectives

Clean and explore the dataset

Filter and analyze specific products or categories

Calculate key statistics:

Average trade amount per product

Most and least popular products

Average rating and wished counts

Create a reusable function to generate product summaries

Document insights and conclusions
How to Use the Notebook

Clone this repository:

git clone https://github.com/YOUR_USERNAME/Pet-Product-Analysis.git


Open the Jupyter Notebook (Pet_Product_Analysis.ipynb) in Jupyter, VS Code, or Google Colab.

Run each cell sequentially to see data analysis, visualizations, and results.

Use the product_summary(product_name) function to quickly get key stats for any product.
